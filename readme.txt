COS221 - PA4

-----------------------------------------------------
BUILDING AND CONNECTING APPLICATION TO DATABASE

1.) Download MariaDB JDBC driver from: https://mariadb.com/kb/en/installing-mariadb-connectorj/
2.) With the project open on Netbeans, go to the "Libraries"-section of the menu on the left side of the screen
3.) Right-click on "Libraries" and click on the "Add jar/folder" option
4.) Select the driver's .jar file that you just downloaded
5.) To build the application, press Shift+F11 with the project open on Netbeans

-----------------------------------------------------
EXECUTING THE APPLICATION

1.)	Go to the folder: COS221_PA4/dist
2.) Run the COS221_PA4.jar file
3.) Enter your MariaDB login details to log in
